.🏠 Boston House Price PredictionThis project implements a high-performance regression model to predict housing prices in Boston. By leveraging the XGBoost Regressor, the model analyzes socio-economic and environmental factors to estimate the median value of owner-occupied homes.📊 Dataset InsightsThe dataset comprises 506 observations across 13 features. The target variable is PRICE (Median value in $1000s).Feature CorrelationBefore modeling, an Exploratory Data Analysis (EDA) was performed to understand the relationships between variables.Key Observations:RM (Rooms): Strong positive correlation with price. More rooms = Higher value.LSTAT (Lower Status %): Strong negative correlation. As the percentage of lower-status population increases, house prices tend to drop.PTRATIO: Higher pupil-teacher ratios correlate with lower property values.⚙️ Project WorkflowData Preprocessing: Cleaned the raw Boston dataset, handled formatting, and verified zero null values.EDA: Visualized feature distributions and multi-collinearity using Seaborn.Feature Engineering: Split the data into Training (80%) and Testing (20%) sets.Modeling: Implemented XGBRegressor with the following hyperparameters:n_estimators: 100max_depth: 2Evaluation: Utilized R-squared ($R^2$) and Mean Absolute Error (MAE) to quantify performance.📈 Model PerformanceThe model demonstrates excellent generalization, explaining over 91% of the variance in the test data.MetricTraining SetTest SetR-Squared ($R^2$)0.9760.916Mean Absolute Error (MAE)1.0921.928Actual vs. Predicted PricesThe scatter plot below illustrates the high alignment between the model's predictions and actual market values.Note: The $R^2$ score is calculated as:$$R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$$🛠 Technical StackLanguage: PythonLibraries: NumPy, Pandas, Matplotlib, SeabornML Frameworks: Scikit-learn, XGBoost
